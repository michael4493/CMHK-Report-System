# Daily Report Tool (外勤員工日報與管理端 Web App)

這是一個為 Project Team 設計的外勤日報工具。目前為 High-fidelity Prototype 階段。

## 🌟 核心功能 (Phase 1-5 完成)
- **Employee Portal**: 響應式日報填寫表單，支援即時歷史紀錄預覽。
- **Management Dashboard**: 三層級 (Project > Domain > Report) 資料展示。
- **Mock AI 總結**: 模擬 Domain 視角的 AI 智能摘要。
- **系統功能**: 支援多語系 (i18n) 與自訂背景，資料全數透過 `localStorage` 暫存，具備完整 CRUD 邏輯。

## 🛠 技術棧
- 純 HTML5 / 原生 JavaScript (Vanilla JS)
- Tailwind CSS (CDN)
